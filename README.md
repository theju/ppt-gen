# PPT Generate

Generate content for your presentations using ChatGPT.

## Install

```
$ git clone https://github.com/theju/ppt-gen.git
$ cd ppt-gen
$ pipenv sync
```

## Usage

```
$ OPENAI_API_KEY=... python server.py
# Visit http://localhost:8080 in your browser
```
